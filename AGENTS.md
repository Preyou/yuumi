# Workspace Bootstrap

Before giving repository-specific advice, making plans, or editing files:

1. Read the root `.ai/INDEX.md`.
2. If the task is mainly inside a workspace or package that has its own `.ai/INDEX.md`, read the nearest local `.ai/INDEX.md` too.
3. Then read only the rules and skills that are relevant to the current task.

Notes:
- Do not rely on `.memory/` or prior conversation context as a substitute for reading `.ai`.
- For tasks in `packages/web/**`, read `packages/web/.ai/INDEX.md` before making repo-specific judgments.
- For tasks in `packages/server/**`, read `packages/server/.ai/INDEX.md` before making repo-specific judgments.
